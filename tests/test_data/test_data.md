# Test Data for libXRK

This contains 2 AIM run sessions, from 2 different cars. Both runs were done at Fuji Speedway.

## SFJ
The SFJ Folder contains data from a run by a beginner in a Super FJ Junior formula car. It contains data from:

* GPS
* Accelerometer
* Gyro
* RPM
* ECT
* Accelerator Pedal Position
* Brake pressure
* Steering Wheel Position


## 86
Contains data from a run by an intermediate driver in a Toyota 86. It has data from:

* GPS
* Accelerometer
* Gyro
* RPM
* ECT
* Oil Temp
* ...and much more
